-------------------------------------------< QuizMaster> ------------------------------------------------------>

QuizMaster is a modern Android app designed to make learning engaging through gamification.
It combines structured quizzes with rewards, progress tracking, and smooth UI to create a real product-level experience.

Features:-

🧠 Smart Quiz Engine
Multiple question types (MCQ, True/False)
Real-time feedback
Countdown timer


Gamified System :-
Earn coins and use them strategically:
💡 Hint → reveal explanation
✂️ 50:50 → eliminate wrong options
⏭️ Skip → move ahead instantly


📊 Progress Tracking :- 
XP & Levels
Daily streaks
Achievements system


Resume Anytime :-
Quiz progress auto-saved using Room
Continue from where you left off


Social Features :- 
Firebase Authentication
Global & monthly leaderboards


Premium UI/UX :- 
Material 3 design
Smooth animations
Haptic feedback
Shimmer loading effects


Tech Stack
Layer	Technology
UI	Jetpack Compose (Material 3)
Architecture	MVVM + Clean Architecture
DI	Hilt
Database	Room + Firestore
Backend	Firebase
Async	Kotlin Coroutines & Flow
Images	Coil
Animations	Konfetti


Project Structure :- 
data/          → repositories, DAO, data sources  
di/            → dependency injection (Hilt)  
presentation/  → UI + ViewModels  
util/          → helper classes  


Setup :- 
git clone https://github.com/mddanisheqbal/Quiz-App
Then:
Open in Android Studio (latest version recommended)
Add your google-services.json inside /app
Sync Gradle
Run the app
